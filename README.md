# MERN Marketplace — Taha

Simple MERN marketplace app. Adapted from the original source (LICENSE kept). I’m adding my own improvements.

## Run locally (dev)

```bash
# --- API ---
cd api
npm install
# create api/.env with:
# MONGO_URI=your-mongodb-connection-string
# JWT_SECRET=your-strong-secret
npm run dev

# --- Client (new terminal) ---
cd client
npm install
npm run dev










```
